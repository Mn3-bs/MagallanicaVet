# MagallanicaVet

**MagallanicaVet** es una página web para una farmacia agro veterinaria, diseñada para ofrecer productos y servicios confiables para el cuidado de animales y del entorno. La plataforma combina un diseño moderno, responsivo y funcional, permitiendo a los usuarios buscar productos, ver detalles destacados y contactar a la farmacia de manera rápida.

---

## Tecnologías utilizadas

- **HTML5** – Estructura y semántica de la página.
- **CSS3** – Estilos y diseño responsivo.
- **JavaScript** – Interactividad y funcionalidades dinámicas.
- **Bootstrap 5** – Framework para componentes y diseño responsivo.
- **Font Awesome & Bootstrap Icons** – Iconografía para mejorar la experiencia visual.

---

## Características principales

- **Navegación clara y responsiva** con menú tipo hamburguesa para móviles y desktop.
- **Sección de contacto rápido** con teléfono y correo electrónico.
- **Carrusel de productos destacados** con descripción y precios.
- **Catálogo de productos** organizado en tarjetas visuales, con opción de agregar a favoritos.
- **Barra de búsqueda** para localizar productos fácilmente.
- **Notificaciones tipo toast** para interacción del usuario (por ejemplo, al agregar productos a favoritos).

---

## Instalación y uso

1. Clonar el repositorio:  
   ```
   git clone <URL_DEL_REPOSITORIO>
   ```
2. Abrir el proyecto en **Visual Studio Code**.  
3. Ejecutar el archivo `index.html` en el navegador.  
4. Navegar y explorar los productos y secciones de la página.
